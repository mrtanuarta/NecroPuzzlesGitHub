# Overview:
NecroPuzzles, a 2D zombie puzzle game where your only goal is to survive. You are stuck at top of the facility that contains four floors and you have to crawl all the way down to the first floor in order to escape. The facility is swarmed by zombies and traps and you have to outsmart their moves to prevent death.

# Gameplay Mechanics: 

## 1. Level Selection

The game contains 4 levels and the player will have to start from level 1. Player's need to beat the level to unlock the next one. Once you've unlocked every level you can choose any level to start from. 

## 2. Difficulty

Each level is harder than the last, with more traps and more zombies that will try to outsmart your moves.

## 3. Escape to Win

Escape through the door while avoiding zombies to beat the game, otherwise you'll be eaten.

# Design Effects:

Customized 8-bit art for characters and environments to give retro vibes.

# Sound Effects:

Dynamic background music & sound effects enhance the experience.

# How to Play:
Use the W, A, S, and D keys to move your character.
Press Buttons to Deactivate Traps.
Press ESC to Pause the game.


# Coding Information:
## - Front-End: JavaFX (Stage, Scene, JButton, Label, Button, EventHandler, MediaPlayer, etc).
Custom 8-Bit Pixel Art for UI & Entities (Player, Zombies, Buttons, etc).

## - Back-End: Java (If-Else, Switch, ArrayList, Try-Catch, etc), Object Oriented Programming (OOP) concepts (Classes, Inheritance, etc), entity management using GamePanel, Level, and entity class structure, and handling all the data & levels using Array.
